# Unity School Simulation 3D

An intelligent 3D school simulation built with Unity featuring autonomous player movement, matrix-based navigation and obstacle avoidance.

#About the Project

This project simulates a school environment in 3D using Unity. The character navigates autonomously through the school using matrix-based pathfinding and avoids obstacles in real time.

# Technologies Used

- **Unity** — 3D game engine
- **C#** — scripting language
- **Matrix-based Navigation** — custom pathfinding algorithm
- **MazeGenerator** — procedural environment generation

##  Features

- Autonomous player movement
- Real-time obstacle avoidance
- 3D school environment simulation
- Matrix-based navigation system

## Project Structure
```
├── script/         # C# scripts
├── Scenes/         # Unity scenes
├── Resources/      # Assets and resources
├── MazeGenerator/  # Navigation logic
└── ProjectSettings/
```

##  How to Run

1. Clone the repository
2. Open with **Unity Hub**
3. Load the scene from `Scenes/` folder
4. Press **Play** in Unity Editor
